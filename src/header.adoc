Main changes in this edition::

- All errors and typos found, including punctuation, layout and
  notation, have been fixed.
- In order to facilitate the searching some entries have been
  splitted, while others have been combined.  {headwords}
- In order to make the contents more legible in any format, and more
  reusable, the tildes in the definitions have been replaced with the
  headwords.
- The notation and the punctuation have been homogenized.
